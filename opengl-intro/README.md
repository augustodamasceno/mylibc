## Brief introduction to OpenGL  
  
## Run compileANDrun.sh  

## Menu  
Button / Function  
### Translation  
  * z:  z--  
  * x:  z = default  
  * c: z++  
  * a: y--  
  * s: y = default  
  * d: y++  
  * q: x--  
  * w: x = default  
  * e: x++

### Rotation  
  * Up/Down: Rotation Y    
  * Left/Right: Rotation X    

### Others  
  * r/R: reset translation and rotation  
  variables to default  
  * m/M: Display Menu   
  * ESC: exit program
